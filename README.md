# AbhiSocial

A full-stack social media web application built using the MERN stack.

## Features

- User Authentication
- Create & Share Posts
- Like / Unlike Posts
- Comment System
- Follow / Unfollow Users
- Cloudinary Image Uploads
- Responsive UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Cloudinary
